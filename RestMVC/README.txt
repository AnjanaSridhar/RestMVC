This is a MAVEN project that provides the solution for the problem provided. 
The folder 'sql' has the queries used to create the tables - Person and Address
The file config.properties under src/main/resources has the database configuration details
 
The link to the home page will be http://localhost:8080/RestMVC/
The login credentials will be admin,admin@123
The hyperlink 'add person' provides the functionality to add a Person to the database.
The home page provides the list of all people in the database.
The link 'View Details' provides details of the person.
The link 'add address' provides the functionality to add a address to a Person

Notes:
1.One Person is allowed to have one address.
2.The function to update a Person or Address is not provided.
